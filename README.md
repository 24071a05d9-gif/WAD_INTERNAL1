# Movie Ticket Booking App

A simple React application for a movie ticket booking website with login, movie list, seat selection, payment, and confirmation pages.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the local URL shown in the terminal.

## App pages

- `/` — Login page
- `/movies` — Movie list page
- `/select-seats` — Choose seats
- `/payment` — Enter payment details
- `/confirmation` — Booking confirmation

## Notes

- Routing is implemented with `react-router-dom`.
- Booking state is managed using React context.
- No backend is included; this is a frontend demo app.
